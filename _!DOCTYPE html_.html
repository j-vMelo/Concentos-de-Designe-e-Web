Affordance

Affordance refere-se às características de um objeto ou interface que sugerem como ele deve ser utilizado. Em design de interfaces, um botão com aparência de ser clicável transmite essa intenção ao usuário. É um conceito central em usabilidade e interação com sistemas, em resumo, é a qualidade de um objeto que permite ao indivíduo identificar sua funcionalidade sem a necessidade de prévia explicação, o que ocorre intuitivamente.

Semiótica

Semiótica é o estudo dos signos e dos processos de significação e comunicação. Em design de interfaces, isso implica em como os elementos visuais e textuais são interpretados pelos usuários. A semiótica ajuda a criar mensagens claras e significativas em uma interface.

Usabilidade

Usabilidade é a medida da facilidade com que uma interface pode ser utilizada. Ela envolve a eficiência, a satisfação e a eficácia com que os usuários conseguem realizar tarefas em um sistema. Uma interface usável melhora a experiência do usuário.

CSS (Cascading Style Sheets)

CSS é uma linguagem usada para descrever a apresentação de documentos HTML. Ele controla o layout, as cores, as fontes e os outros aspectos visuais da página, tornando-a mais atraente e funcional.

Formas de Incluir CSS

Existem três formas de incluir CSS em um documento HTML:

  • Inline: CSS é aplicado diretamente em um elemento HTML através do atributo style. Recomendado para ajustes rápidos em elementos específicos.
  • Interno: CSS é colocado dentro da tag <style> no cabeçalho do documento. Ideal para estilos específicos de uma página.
  • Externo: CSS é armazenado em um arquivo separado e vinculado ao documento HTML. Recomendado para grandes sites, pois permite reutilizar os estilos em várias páginas.
